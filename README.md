### Simple My Twitter Post Web App that uses python Flask and Tweepy <br><br>

Dependent(must install)
- Flask
- Python-Dotenv
- Tweepy
- Requests

<br><br>

To Run The Code<br>
`python3 -m venv venv && source venv/bin/activate` <br>
`pip install flask python-dotenv tweepy requests`<br>
`python3 app.py`
